## Projects

### 🛒 [E-Commerce Backend](https://github.com/ravalika-reddy04/ECommerceBackend)
REST API for an online marketplace — product catalog, shopping cart, order checkout with stock management, and payment simulation.
- Multi-role: Admin, Seller, Customer
- Cart → Order checkout atomically deducts stock and generates payment record
- 30 unit and controller tests (JUnit 5 + Mockito + MockMvc)
- **Stack:** Spring Boot 3 · Java 21 · PostgreSQL · JWT · Flyway · Docker · GitHub Actions

### 📋 [Leave Management System](https://github.com/ravalika-reddy04/LeaveManagementSystem)
Backend API for employee leave request lifecycle — submission, manager approval/rejection, and automatic balance tracking.
- Working-day calculation excluding weekends; overlap detection prevents conflicting requests
- Leave balance tracked per user, per leave type, per year with used/pending/available breakdown
- RFC 9457 ProblemDetail error responses; Swagger UI for interactive testing
- **Stack:** Spring Boot 3 · Java 21 · PostgreSQL · JWT · RBAC · Flyway · Docker

### 🚗 [Vehicle Service Management System](https://github.com/ravalika-reddy04/VehicleServiceManagementSystem)
REST API for automotive service centers — customer/vehicle registry, appointment booking with conflict detection, service record lifecycle, and technician assignment.
- Service record status machine: Scheduled → In Progress → Completed / On Hold / Cancelled
- Appointment conflict detection prevents double-bookings
- **Stack:** Spring Boot 3 · Java 21 · PostgreSQL · JWT · RBAC · Flyway · Docker
